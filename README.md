## Major Projects

### Reusable Launch Vehicle (RLV) with VTOL Capability
Design and fabrication of a reusable launch vehicle prototype capable of **Vertical Take-Off and Landing (VTOL)**.

Key Features:
- Vertical launch and controlled landing concept
- Real-time telemetry monitoring system
- GPS-based tracking system
- Embedded flight data transmission
- Structural design and fabrication of the airframe
- Integration of onboard avionics and sensors

---

### Rocket Propulsion – KNSB Solid Rocket Motor
Experimental rocketry project focused on **solid rocket propulsion using KNSB propellant (Potassium Nitrate + Sorbitol).**

Key Features:
- Solid rocket motor design
- Propellant grain casting and testing
- Combustion chamber design
- Rocket propulsion performance analysis
- Small-scale rocket flight experiments

---

### UAV / RC Aircraft Systems
Design and development of **RC aircraft platforms for surveillance and monitoring applications.**

Key Features:
- RC MQ-9 Reaper inspired surveillance aircraft
- Camera-based aerial monitoring system
- Flight stability testing
- Aerodynamic design and optimization
- Team project winner at engineering project expo

---

### CubeSat Telemetry Dashboard
Development of a **CubeSat-inspired environmental monitoring system with web telemetry dashboard.**

Key Features:
- ESP32 based telemetry system
- Real-time sensor data monitoring
- Web dashboard for live environmental data
- Temperature, pressure, UV and battery monitoring

---

### Aircraft CAD Modelling
High-precision aircraft CAD modelling based on real aircraft blueprints.

Models Created:
- Boeing 737
- ATR-72 Regional Turboprop
- RC Aircraft structures

Tools Used:
- SolidWorks
- Fusion 360
